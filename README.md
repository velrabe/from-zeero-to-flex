# From Zero to Flex

Игровой прототип idle-кликера для Google Play.

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
npx serve .
```

## Структура

- `index.html` — разметка экранов
- `styles.css` — простая верстка, 16:9 контейнер, без цветов и анимаций
- `script.js` — логика игры (баланс, бизнесы, апгрейды)

## Формат экрана

- Основной контейнер: 16:9, центрирован
- Высота привязана к `100vh`, ширина вычисляется автоматически
- Визуальный стиль упрощен до системного шрифта, белого фона, черного текста и простых рамок

## Экранa

1. **Home** — баланс, доход/сек, кнопки «Начать работу» и «Купить бизнес»
2. **Мои бизнесы** — карточки купленных бизнесов, переход в апгрейды
3. **Купить бизнес** — магазин, карточки с прогрессией
4. **Инфо о бизнесе** — описание, стоимость, доход, кнопка «Купить»
5. **Апгрейды** — список улучшений внутри бизнеса

## Стартовый баланс

100 монет для тестирования первой покупки (Лимонный ларек — 80).

## Публикация в GitHub Pages

Репозиторий настроен для публикации текущей статической версии через GitHub Actions. Workflow `.github/workflows/github-pages.yml` запускается при пуше в ветки `work`, `main` или `master`, а также вручную через `workflow_dispatch`.

После первого пуша включите Pages в настройках репозитория: **Settings → Pages → Build and deployment → Source → GitHub Actions**.
